# Portfolio

Static personal site with three pages (home, about, projects) and organized assets.

## Structure
- `index.html`, `about.html`, `project.html` – page entry points
- `assets/css` – page-specific stylesheets
- `assets/images` – icons, screenshots, and logos
- `assets/docs` – downloadable documents (resume)
- `CNAME` – custom domain config for GitHub Pages

## Local preview
Open `index.html` in a browser or use a simple static server (e.g., `python -m http.server` in the repo root).
